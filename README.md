## CURRENCY-CONVERTER 💵💰

This is a currency-converter app that let the user enter amount and choose which currency he wanted to convert it .I use free API here , Exchangerates-free API.

------------------------------

## FEATURES ✨

- Enter amount
- from which currency
- to which currency
- Convert button
- A result div that display result after conversion.
- Exchange rates free API.

--------------------------

## TECHNOLOGY

1. **HTML**
2. **CSS**
3. **JAVASCRIPT**
4. FREE API EXCHANGE RATE
